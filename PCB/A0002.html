Full Feature Flight Computer

Description

The full feature flight computer is a complete solution for parachute ejection and data logging of high power rocketry flights. The computer is equipped with the base flight computer features such as a barometric pressure sensor for measuring altitude, two pyro ports for dual-deploy recovery, and a USB interface for programming. In addition to these base features, the full feature flight computer contains an IMU for full state measurement using an accelerometer, gyroscope, and magnetometer. The board also contains a high-G accelerometer in order to measure high accelerations during the rocket boost phase. To support staging and redundant ejection setups, the board is equipped with two auxiliary pyro ports that are software configurable.

Technical Specifications

Dimensions: 1.2"x2.9"

Microcontroller: STM32H750VBT6

Processor: ARM Cortex-M7

Maximum Clock Speed: 480 MHz

Memory: 128 kB Flash, 1MB RAM, 4 Mb External Flash

Supply Voltage: 3.6-16V

Maximum Current Draw: 1A

Sensors: Barometric Pressure Sensor, IMU, High-G Accelerometer

Connectivity: USB
