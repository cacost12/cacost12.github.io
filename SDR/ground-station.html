Ground Station Computer

Description

The ground station computer is a raspberry-pi based computer which serves as a universal user interface for all of Sun Devil Rocketry's avionics systems. Currently, the computer is being used mainly with the liquid propulsion avionics system and the flight computer, but support for the hybrid and solid propulsion groups will be eventually added once the projects are more developed. Shown above is the liquid engine GUI, which is used to control the avionics system for a liquid rocket engine. The GUI is Tk based and written entirely in Python. Communication between the GUI and the liquid engine controller is established through a USB connection between the raspberry-pi and an XBee wireless module.
